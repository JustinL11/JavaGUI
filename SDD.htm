Purpose:

This program serves to take in a user's text input and return the amount of time recurring words appear.

This program is written in Java 16.0.2, there is a chance for error if ran in new variation. Program prompts a GUI interface to which the user chooses a text file to frequency sort. An outfile is created with the frequency results.

Testing

Junit5 on version 1.9.2 Testing wheter or not our Hash function properly works as it does all of the sorting. JUnit Jar is provided within the package for testing functions.
